# Privacy Policy for Ghost SEO Generator

**Last updated:** 07 January 2026

## 1. Overview

Ghost SEO Generator is a browser extension designed to help sellers on Wildberries and Ozon create SEO-optimized product descriptions using artificial intelligence. We are committed to protecting your privacy and ensuring transparency about how our extension handles data.

## 2. What Data We Collect

Ghost SEO Generator **does not collect, store, or transmit** any personal data. 

The extension only reads the following data from the product page you are currently viewing on Wildberries or Ozon:

- Product name
- Product description
- Product characteristics (size, color, material, etc.)
- Product price

This data is read **only when you manually click** the "Generate SEO description" button.

## 3. How Data is Used

When you click the button:

1. **Product data** is read from the current product page
2. **This data is sent** to DeepSeek API (a third-party AI service) to generate a unique SEO-optimized description
3. The generated text is displayed in the popup window
4. You can then copy the text to your clipboard

**Important notes:**
- No data is saved or stored anywhere
- No data is shared with anyone except DeepSeek API (for generation)
- No data is used for any purpose other than generating your SEO description
- All actions are initiated by you (no background data collection)

## 4. Third-Party Services

### DeepSeek API
The extension sends product data to DeepSeek API to generate descriptions. DeepSeek's data handling is governed by their own privacy policy:
- [DeepSeek Privacy Policy](https://platform.deepseek.com/privacy-policy)

**What is sent to DeepSeek:**
- Product name, description, characteristics, price
- Platform name (Wildberries or Ozon)

**What is NOT sent:**
- No personal information
- No login credentials
- No browser history
- No cookies or tracking data

## 5. Permissions Used

### activeTab
Used to read product data from the current page **only when you click the generate button**.

### storage
Used to save your preference (extension enabled/disabled state) between browser sessions.

### host_permissions
- `wildberries.ru` and `ozon.ru` — to access product data
- `api.deepseek.com` — to send data for SEO generation

All permissions are used strictly for the extension's single purpose.

## 6. Data Security

- All data transmission to DeepSeek API is encrypted (HTTPS)
- No data is stored locally or remotely
- No data logs are maintained

## 7. Your Rights

Since we do not collect any personal data:
- We have no data to delete
- We have no data to export
- We cannot identify you through our extension

## 8. Children's Privacy

Ghost SEO Generator does not knowingly collect any personal information from children under 13 years of age.
